## ðŸŽ­ Character Interaction Modes

Expand beyond the current "Improv Scene" into distinct interaction paradigms:

### 1. Debate Mode

- Two agents argue opposing sides of a topic
- Director assigns positions and moderates
- Score/voting system for "winner"

### 2. Interview Mode

- One agent plays host, others are guests
- Structured Q&A format
- Guest can evade, deflect, or overshare based on personality

### 3. Roast Mode

- Agents take turns roasting each other
- Crowd reaction sounds/visuals
- "Too far" detector that triggers apologies

### 4. Story Time Mode

- Collaborative storytelling
- Each agent adds to a narrative
- Genre selector (horror, romance, sci-fi, absurdist)

### 5. Therapy Session Mode

- One agent plays therapist
- Others share "problems"
- Comedic misunderstandings and bad advice

### 6. Game Show Mode

- Trivia, word games, or improv games
- Points system
- Sound effects and fanfare
